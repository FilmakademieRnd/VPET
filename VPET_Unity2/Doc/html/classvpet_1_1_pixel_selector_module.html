Detailed Description

Module to be used per camera that provide selection from main camera. There can be multiple instances of this class, providing local camera space selection.

◆ GetSelectableMaterial()

Material vpet.PixelSelectorModule.GetSelectableMaterial ( Material  material)
private

Gets a cached adjusted material or creates a new one based on the specified material. Selectable materials are identical to the specified material except that they have the SelectableType tag set so they are rendered in the replacement pass used to render selectable ids.

Note that all adjusted materials are destroyed when the selection manager is destroyed!

Parameters
materialThe material to be changed for selection rendering.
Returns
An adjusted m_instance of the specified material with the selectable tag set.
