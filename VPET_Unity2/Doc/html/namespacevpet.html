Detailed Description

This class colorizes the object according to the light color and scales the object based on the distance to the camera. It is beeing applied / should be attached on gameObjects visualizing a directional light.

This class colorizes the object according to the light color and scales the object based on the distance to the camera, the range of the light and the coneAngle. It is beeing applied / should be attached on gameObjects visualizing a spot light.

This class rotates the light icon sprite to always face towards to camera and and scales them by the distance to the camera. This class is beeing applied / should be attached on each light icon quad attached to any light source.

This class colorizes the object according to the light color and scales the object based on the distance to the camera and the range of the light. It is beeing applied / should be attached on gameObjects visualizing a point light.
